# Eagle

## An Android Bitcoin xPub/Address transaction viewer

Using latest recommended practices for Android app development.

**Jetpack**
* Navigation
* Lifecycle
* ViewModel
* LiveData

**Reactive**
* RxJava
* RxKotlin
* RxAndroid
* RxBinding

**Dependency Injection**
* Hilt
* Dagger 2

**Network**
* OkHttp
* Retrofit

**UI Components**
* Material
* FastAdapter
